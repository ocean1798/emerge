# 08 Known Issues

## MVP 发布已知问题

本文档记录 Emerge Web MVP 发布时的已知问题和限制。这些问题不影响核心功能使用，但用户应了解。

---

## 高优先级 (P0)

### 1. API Key 需要每次启动时重新输入

**问题描述**：API key 只在当前 API 进程内存中保存，重启后需要重新配置。

**影响范围**：所有需要 LLM 功能的用户（Ask、智能问答）

**临时解决方案**：
- 在 `server/.env.local` 中配置 `OPENAI_API_KEY`，API 启动时会自动读取
- 或在产品内"模型设置"中每次启动后重新输入

**后续计划**：未来版本将接入操作系统 Keychain 或加密本地密钥库

---

### 2. URL 抓取仅支持公开静态页面

**问题描述**：URL 捕获功能使用启发式 HTML 抽取，无法处理：
- 需要登录的页面
- 反爬虫保护的页面
- 需要 JavaScript 渲染的单页应用 (SPA)
- 动态加载内容的页面

**影响范围**：URL 捕获功能

**临时解决方案**：对于无法自动抓取的页面，可手动复制内容后使用"新建笔记"功能

**后续计划**：未来版本将支持 headless browser 渲染和更多反爬策略

---

## 中优先级 (P1)

### 3. 原始内容预览仅显示索引片段

**问题描述**：对象内容预览显示的是进入索引的文本片段，而非完整原文。对于长文档，只能看到部分内容。

**影响范围**：对象预览功能

**临时解决方案**：点击"查看源文件"可查看完整原文

**后续计划**：未来版本将提供完整的原文阅读器

---

### 4. 桌面应用尚未打包

**问题描述**：当前 MVP 仅支持浏览器访问，Tauri 桌面壳尚未完成打包和分发。

**影响范围**：希望使用桌面应用的用户

**临时解决方案**：通过浏览器访问 `http://127.0.0.1:5173/` 使用

**后续计划**：桌面应用将在后续版本发布

---

### 5. Embedding 模型需要本地安装 Ollama

**问题描述**：向量检索功能依赖本地运行的 Ollama 服务和 embedding 模型。如果 Ollama 未安装或未运行，系统会回退到词法检索。

**影响范围**：语义搜索和 Ask 功能的准确性

**临时解决方案**：
1. 安装 Ollama：https://ollama.ai
2. 拉取 embedding 模型：`ollama pull embeddinggemma`
3. 确保 Ollama 在 `http://localhost:11434` 运行

**后续计划**：未来版本可能支持更多 embedding provider

---

### 6. 批量导入大文件可能较慢

**问题描述**：导入大文件（>1MB）或批量导入多个文件时，处理时间可能较长，且没有进度条。

**影响范围**：文件导入功能

**临时解决方案**：耐心等待处理完成，或分批导入较小的文件

**后续计划**：未来版本将添加进度指示和后台处理

---

## 低优先级 (P2)

### 7. 移动端布局为简化版

**问题描述**：移动端采用堆叠布局，部分功能（如三栏工作台）在小屏幕上体验受限。

**影响范围**：移动端用户

**临时解决方案**：建议在桌面端使用以获得最佳体验

**后续计划**：未来版本将优化移动端交互

---

### 8. 国际化支持有限

**问题描述**：当前支持简体中文和英文，但部分新增功能的翻译可能不完整。

**影响范围**：英文用户

**临时解决方案**：切换到简体中文查看完整文案

**后续计划**：持续完善多语言支持

---

### 9. 本地数据存储为单 JSON 文件

**问题描述**：所有数据存储在 `server/local-data/store.json` 单个文件中，大量数据时可能影响性能。

**影响范围**：数据量较大的用户

**临时解决方案**：定期清理不需要的对象

**后续计划**：未来版本可能迁移到 SQLite 或其他本地数据库

---

## 技术限制

### 10. API 仅监听 localhost

**问题描述**：本地 API 默认只监听 `127.0.0.1:8787`，不支持局域网访问。

**影响范围**：希望在多设备间共享的用户

**临时解决方案**：如需局域网访问，可修改 `server/src/index.js` 中的监听地址

**后续计划**：未来版本将支持配置监听地址和认证

---

### 11. 无数据导出/备份功能

**问题描述**：当前没有批量导出所有数据或创建备份的功能。

**影响范围**：数据迁移和备份

**临时解决方案**：手动复制 `server/local-data/` 目录

**后续计划**：未来版本将添加数据导出和备份功能
